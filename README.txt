Server
-------

The server uses a PostgreSQL database, meaning that in order to run the server this will need to be installed to your PC.

To run the server:
1. Enter the server folder using the cd command

2. Install node_modules by using the command 'npm i'

3. Once installed, run the server using 'npm run start'

To run the unit/integration tests:
1. Enter the server file using the cd command

2. Install node_modules by using the command 'npm i'

3. Once installed, run the server using 'npm run test'

UI
-------
To run the UI:
1. Enter the UI folder using the cd command

2. Install node_modules by using the command 'npm i'

3. Once installed, run the UI using 'npm run serve'
